## 🚀 安装步骤（blackboy操作）

### 第一步：让王颖填API密钥

让她填写 `API配置/密钥收集表.txt`，至少需要：
- DeepSeek API Key（必须）
- Zotero API Key（必须，用于文献管理）
- 其他可选

### 第二步：帮她装WSL2

按 `安装指南/01-WSL2安装指南.md` 操作，预计30分钟。

### 第三步：装Hermes

按 `安装指南/02-Hermes安装与配置.md` 操作，预计20分钟。

### 第四步：注入小腾子灵魂

```bash
# 复制SOUL.md到Hermes配置目录
cp "三省六部指令体系/SOUL.md" ~/.hermes/SOUL.md

# 复制MEMORY模板
cp "学术能力包/MEMORY-学术版.md" ~/.hermes/MEMORY.md

# 复制进化日志
cp "三省六部指令体系/小腾子进化日志.md" ~/.hermes/小腾子进化日志.md
```

### 第五步：运行环境配置脚本

```bash
# 一键安装学术工具链
chmod +x scripts/setup_academic_env.sh
./scripts/setup_academic_env.sh
```

### 第六步：注入技能包

```bash
# 复制所有技能到Hermes技能目录
cp -r skills/* ~/.hermes/skills/
```

### 第七步：配置API密钥

```bash
# 按API配置清单配置
hermes config set providers.deepseek.api_key "她的密钥"
```

### 第八步：测试

```bash
# 启动Hermes
hermes chat

# 测试小腾子
> 你好，小腾子
> 帮我查一下Thellier方法的基本原理
> 朝议：我的课题该怎么规划？
```

## 🏛️ 小腾子核心特色

### 1. 主动思考

小腾子不是被动等命令的。它会：
- **主动预判**：你提到课题，它会自动联想到相关文献、方法、工具
- **主动建议**：完成任务后，它会建议下一步做什么
- **主动学习**：它会记住你的习惯，越用越懂你

### 2. 小腾子笔记

每次完成任务后，小腾子会给你一个笔记：

```
📝 小腾子笔记
━━━━━━━━━━━━
🔍 本次总结：完成了10个样品的退磁数据分析
💡 我发现：您的样品在300-500℃区间有明显的磁化强度下降
🎯 建议下一步：
  1. 对这批样品做岩石磁学分析
  2. 检查是否存在次生磁化
  3. 与杨佳豪(2022)的数据对比
🔄 进化记录：记住了您的样品命名规则（PL-XX）
```

### 3. 三省六部治理

- **口谕**：日常对话，小腾子直接回答
- **圣旨**：复杂任务，走三省流程
- **朝议**：多角度分析问题
- **封驳**：方案有问题时，小腾子会委婉指出
- **起居注**：查看工作进度

### 4. 自动进化

小腾子会自动：
- 记住你的偏好
- 学习你的工作习惯
- 总结经验教训
- 改进自己的回答

## ⚠️ 注意事项

1. **API密钥安全**：不要在对话中明文输入密钥，使用环境变量或配置文件
2. **数据备份**：重要数据定期备份，小腾子不会自动备份
3. **网络环境**：WSL2可能需要配置代理，详见安装指南
4. **模型选择**：DeepSeek适合日常任务，复杂推理用MiMo
5. **进化需要时间**：小腾子需要1-2周才能真正懂你
